навыки

HTML 5

Моя разметка семантична и структурирована. Знакома с препроцессором Pug, но редко его использую.

CSS 3

Моя любимая часть верстки. Стили всегда разбиты по компонентам. Избегаю большой вложенности селекторов.

JS

Считаю свой уровень знания JS отличным, чтобы справиться с работой верстальщика.

БЭМ

Работаю по данной методологии достаточно давно, поэтому редко сомневаюсь в наименовании класса.

SASS

С удовольствием использую для облегчения работы со стилями. Также был опыт использования Stylus.

Jquery

Знаю не хуже, чем нативный JS. Использовала Jquery в большей части своих проектов.

Gulp

Gulp - мои руки и ноги, поэтому научилась дорабатывать и редактировать свой сборщик.

React

Уровень знания этой библиотеки считаю стартовым. Использовала в связке с Redux для тренировочных проектов.

Photoshop

Очень часто использую Adobe Photoshop при работе с макетами, поэтому привыкла к данной программе. Но мне не составит труда освоить что-то новое
